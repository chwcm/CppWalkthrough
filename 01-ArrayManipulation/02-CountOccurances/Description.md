Ask for integer numbers until 0 is given.
You can expect the numbers to be non-negative.
Print out how many times the numbers 1-5 appeared, and how many times were a number greater than 5 given.

Example input:
```
1
2
3
4
3
6
8
3
9
3
4
0
```

Example output:
```
1: 1
2: 1
3: 4
4: 2
5: 0
larger: 3
```
